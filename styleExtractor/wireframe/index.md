| 名稱 | Mobile | Desktop |

| 名稱 | 寬度 |
|------|------|
| sm (mobile) | 預設 |
| md (tablet) | `≥ 768px` |

| 用途 | Mobile | Desktop |

### 過渡動畫

| 名稱 | 值 |

| 用途 | 值 |
|------|-----|
| 卡片默認 | 無 (無陰影，乾淨風格) |

### 區塊分隔風格

各首頁 Section 之間使用 SVG 弧形曲線分隔，而非直線。弧形高度：
- 手機: `3.38rem`
- 桌面: `9.69rem`

配色對照：白 ↔ 米色(`#fbf7ef`) ↔ 淡綠(`#f5f9e9`) ↔ 白 交替
